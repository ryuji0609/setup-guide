🚀 導入マニュアル

AI Threads Sorterをあなたの環境で動かすための3ステップ

このツールは、「あなた専用のGoogleスプレッドシート」と連携して動きます。
セキュリティを守りながら自動投稿を行うために、最初の1回だけ以下の設定をお願いします！

1

専用スプレッドシートをコピーする

まずは、ポストを保存・管理するための「あなた専用のデータベース（スプレッドシート）」を作成します。

  1. 以下のボタンをクリックして、テンプレートを開きます。
    2. 📊 テンプレートを開く
  2. 左上のメニューから 「ファイル」 ＞ 「コピーを作成」 をクリックします。
    3. [ここに「ファイル＞コピーを作成」のスクショ画像を入れると親切です]
  3. 名前を「設定用：AI Threads Sorter」など分かりやすいものにして、「コピーを作成」ボタンを押します。
  4. これで、あなたのGoogleドライブに専用のシステムが保存されました。

✓ ここまでできたら、新しく開いた「コピーされた方の画面」でStep 2に進みます。

2

システム（GAS）を起動してURLを発行する

次に、ツール（画面）とスプレッドシートを繋げるための「専用URL」を発行します。プログラミングの知識は一切不要です！

  1. コピーしたスプレッドシートの上のメニューから 「拡張機能」 ＞ 「Apps Script」 をクリックします。
  2. 新しい画面（プログラムの画面）が開いたら、右上の青い 「デプロイ」 ボタンを押し、「新しいデプロイ」 を選びます。
    3. [ここに「デプロイ＞新しいデプロイ」のスクショ画像を入れると親切です]
  3. 設定画面が出たら、以下の通りに設定してください：
    • 種類の選択（左の歯車マーク）: ウェブアプリ
    • 説明: （空欄でOKです）
    • 次のユーザーとして実行: 自分（あなたのメールアドレス）
    • アクセスできるユーザー: 全員 ←ここが一番重要です！
  4. 設定できたら右下の 「デプロイ」 をクリックします。
⚠️「アクセスを承認」と出た場合
Googleのセキュリティ警告です。以下の手順で許可してください：
「アクセスを承認」をクリック ＞ 自分のGoogleアカウントを選択 ＞ 左下の「詳細」をクリック ＞ 一番下の「（安全ではないページ）に移動」をクリック ＞ 「許可」をクリック。
  1. 完了画面に表示される ウェブアプリのURL の下にある「コピー」ボタンを押します。
    2. [ここに「URLをコピー」の完了画面のスクショ画像を入れると親切です]

    ※コピーしたURL（https://script.google.com/macros/s/.../exec）は、あとで使うのでメモ帳などに貼り付けておいてください。

3

自動投稿タイマー（トリガー）をセットする

最後に、スプレッドシートに保存されたポストが「自動的にThreadsに投稿される」ように、タイマーをセットします。

  1. 先ほどデプロイした画面（Apps Script）のまま、左側のメニューから 時計のマーク（トリガー） をクリックします。
  2. 右下の青い 「＋ トリガーを追加」 をクリックします。
    3. [ここに「トリガーを追加」ボタンのスクショ画像を入れると親切です]
  3. 以下の通りに設定して「保存」を押します：
    • 実行する関数を選択: checkScheduledPosts
    • 実行するデプロイを選択: Head
    • イベントのソースを選択: 時間主導型
    • 時間ベースのトリガーのタイプを選択: 分ベースのタイマー
    • 時間の間隔を選択（分）: 10分おき
    4. [ここに上記5項目の設定画面のスクショ画像を入れると親切です]

🎉 お疲れ様でした！これで裏側の準備はすべて完了です！

4

ツールを使ってみよう！

いよいよツールへログインします！

  1. 以下のボタンからツール画面を開きます。
    2. ✨ ツールを開く
  2. 合言葉 「【※合言葉※】」 を入力してログインします。
  3. 「✨ AI 自動生成設定」を開き、Step 2でコピーした「Google Apps Script デプロイURL」 を貼り付けます。
  4. Gemini API キーと基本プロフィールを入力して、「AIでポストを生成する」を押せば完了です！

生成されたカードはスワイプで仕分けでき、右に「保存（Keep）」すると、先ほど作ったあなたのスプレッドシートに自動的に送信・保存されます！
